# Student Performance Predictor

A simple Machine Learning project that predicts a student's exam score based on the number of study hours using Linear Regression.

## Overview

This project demonstrates the fundamentals of supervised machine learning by training a Linear Regression model on a small dataset and predicting student performance.

## Features

- Predict exam scores from study hours
- Built using Python and Scikit-learn
- Simple and beginner-friendly implementation
- Demonstrates basic Machine Learning workflow

## Technologies Used

- Python
- Pandas
- NumPy
- Scikit-learn

## Installation

Clone the repository:

```bash
git clone https://github.com/YOUR_USERNAME/student-performance-predictor.git
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the project:

```bash
python src/main.py
```
